# SDMR-NATCOR-2026

This repo contains Supporting material for NATCOR Course on **System Dynamics Modelling using R**, July 2026.

There are five objectives of the initial workshop, which are to:

* Understand data frames and plot simulation data using ggplot2
* Analyse sensitivity runs with dplyr
* Build a model using deSolve
* Run a sensitivity sweep with purrr, deSolve and dplyr
* Analyse sensitivity runs using Explanatory Model Analysis, and apply these to an SIR Model with Hospitalisations


To run the code, it is recommended to setup an account on posit ([posit.cloud (formerly known as RStudio Cloud)](https://posit.cloud/)), and then use the option **New Project from Git Repository**, and enter https://github.com/JimDuggan/SDMR-NATCOR-2026.

Once the project is created, open the file `install_packages.R` in the `setup` folder, and click on the button called "Source", as this will run the file.

Any of the R files should then run ok.
